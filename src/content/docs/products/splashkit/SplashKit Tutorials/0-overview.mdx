The SplashKit tutorials team works on the creation of tutorials using the SplashKit library. The team is responsible for creating tutorials that cover a wide range of topics, from Graphics, Interfaces, Networking and Raspberry Pis. The tutorials are designed to help users learn how to use SplashKit and to provide them with the knowledge they need to create their own projects. Each tutorial contains an explanation of the topic, the steps involved, as well as code snippets done in C#, C++, Python and Pasal.

## Tutorial Topics

The current list of tutorial topics includes:

- Animations
- Audio
- Camera
- Graphics
- Input
- Interface
- JSON
- Networking
- Raspberry GPIO
- Resource Bundles
- Sprites
- Utilities
